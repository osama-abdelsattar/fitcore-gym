# FitCore Gym - Landing Page

FitCore Gym is a modern, conversion-focused landing page designed to showcase gym facilities, display member transformations, and convert visitors into active members with an intuitive user experience.

## Features

- **Hero Section:** Bold introduction with striking imagery and clear Call-to-Action (CTA) buttons to drive engagement.
- **Features & Services:** Clean grids showcasing gym strengths (24/7 access, top equipment, certified trainers) and secondary perks like group classes.
- **Social Proof:** Dedicated testimonials and real-time statistics to build trust and credibility.
- **Interactive Membership Form:** A responsive multi-field form allowing users to select membership tiers (Basic, Premium, VIP) and submit fitness goals.
- **Fully Responsive:** Styled using modern layout techniques to look flawless on mobile, tablet, and desktop screens.

## Tech Stack

- **Structure:** HTML5 (Semantic elements)
- **Styling:** CSS3, Flexbox
- **Typography:** Google Fonts (Inter)

## Project Structure

- `index.html` - The core markup structure of the landing page.
- `css/main.css` - Main style for the page which uses "Float" for layout.
- `css/flex.css` - Custom stylesheet built using Flexbox for flexible and fluid responsive grids.
